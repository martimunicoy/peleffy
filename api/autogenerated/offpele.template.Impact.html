offpele.template.Impact¶

class offpele.template.Impact(molecule)[source]¶

It is in charge of writing a Molecule object as a PELE’s Impact template.

__init__(molecule)[source]¶

Initializes an Impact object.

Parameters

molecule (An offpele.topology.Molecule) – A Molecule object to be written as an Impact file

Examples

Write the Impact template of a offpele’s molecule

>>> from offpele.topology import Molecule
>>> from offpele.template import Impact

>>> molecule = Molecule('molecule.pdb')
>>> molecule.parameterize('openff_unconstrained-1.1.1.offxml')
>>> impact = Impact(molecule)
>>> impact.write('molz')

offpele.topology.Dihedral¶

class offpele.topology.Dihedral(index=- 1, atom1_idx=None, atom2_idx=None, atom3_idx=None, atom4_idx=None, periodicity=None, prefactor=None, constant=None)[source]¶

It represents a dihedral in the topology.

It can be a proper or an improper dihedral.
